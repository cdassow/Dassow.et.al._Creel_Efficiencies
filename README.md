# Dassow.et.al._Creel_Efficiencies
Companion to Dassow et al. paper on creel efficiencies. Contains code and data to reproduce the analyses in this paper.

The modeling component of this analysis uses the `BayesianTools` R package found here: https://cran.csiro.au/web/packages/BayesianTools/index.html

Those unfamiliar with this package are highly encouraged to use the vignette provided here: https://cran.csiro.au/web/packages/BayesianTools/vignettes/BayesianTools.html
